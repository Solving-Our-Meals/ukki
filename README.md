## Project Name (프로젝트 이름)
### Ukki (우끼)
    - 우리들의 끼니 해결의 줄임말을 사용
    - 사용자에게 너무 무겁게 다가가지 않도록 의성어 느낌도 받을 수 있도록 프로젝트 이름이자 홈페이지 이름을 정함
